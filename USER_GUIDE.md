# 🌳 匿名情绪树洞 - 用户指南

**欢迎来到匿名情绪树洞！**  
这里是一个温暖的倾诉空间，让每一份心事都有处安放，让每一份温暖都触手可及。

## ✨ 树洞是什么？

匿名情绪树洞是一个**安全、温暖、完全匿名**的在线倾诉平台。在这里，你可以：

- 📝 **匿名分享**心事、烦恼、快乐或迷茫
- 💌 **获得温暖**回应，感受被理解的温暖
- 🌱 **帮助他人**，你的故事可能给他人带来力量
- 🛡️ **完全保密**，所有信息都会严格匿名处理

## 🚀 快速开始

### 访问树洞
- **官方网站**: https://secret-treehouse.vercel.app
- **留言展示页**: https://secret-treehouse.vercel.app/messages.html
- **API测试页**: https://secret-treehouse.vercel.app/test-api.html

### 第一次使用
1. 打开树洞首页
2. 浏览他人的温暖故事
3. 点击"写下你的心事"开始分享
4. 选择情绪，写下想说的话
5. 提交后等待温暖回应

## 📝 如何分享心事

### 步骤1：选择情绪
树洞提供8种情绪标签，帮助你表达此刻的心情：

| 情绪 | 表情 | 适合场景 |
|------|------|----------|
| 😊 快乐 | 分享喜悦、感恩、小确幸 | 今天遇到的好事，感恩的时刻 |
| 😢 悲伤 | 表达难过、失落、思念 | 失去重要的人或物，感到孤独 |
| 😰 焦虑 | 描述紧张、担心、压力 | 工作压力、考试焦虑、未来迷茫 |
| 😫 压力 | 倾诉负担、疲惫、 overwhelmed | 工作太多、生活压力大 |
| 🤔 迷茫 | 表达困惑、选择困难 | 人生十字路口，不知道方向 |
| 🌙 孤独 | 描述寂寞、隔阂感 | 即使身边有人也觉得孤独 |
| 😌 平静 | 分享宁静、平和时刻 | 内心的平静，小确幸时刻 |
| 🙏 感恩 | 表达感谢、珍惜 | 感恩健康、家人、机会 |

### 步骤2：写下你的故事
- **内容**: 10-1000个字符，真实表达即可
- **昵称**（可选）: 可以给自己起个喜欢的名字，留空则显示"匿名朋友"
- **分类**: 选择最相关的分类，帮助树洞更好理解
- **公开设置**: 选择是否愿意公开分享帮助他人

### 步骤3：提交等待回应
- ✅ 提交成功后会收到确认提示
- ⏳ 温暖回应通常在24小时内
- 🔄 可以随时回来查看回应

## 🛡️ 隐私和安全

### 我们的承诺
- 🔒 **严格匿名**: 所有个人信息都会去除
- 🚫 **不追踪**: 不收集任何能识别身份的信息
- 📊 **数据加密**: 所有数据安全存储
- 👁️ **审核机制**: 所有内容都会人工审核

### 你的隐私
- 昵称可以随时更改
- 可以选择不公开故事
- 可以随时删除自己的故事（联系管理员）
- 所有数据你都有完全控制权

## 💌 温暖回应系统

### 谁在回应？
- 🌳 **树洞团队**: 经过培训的温暖志愿者
- ❤️ **专业建议**: 基于心理学和咨询原则
- 🌟 **温暖陪伴**: 重点是倾听和理解，不是评判

### 回应原则
1. **非评判性**: 不批评、不指责、不建议
2. **共情理解**: 尝试理解你的感受和处境
3. **温暖支持**: 提供情感支持和鼓励
4. **资源指引**: 在需要时提供专业资源信息

### 回应时间
- ⏰ **通常**: 24小时内
- 🚀 **快速通道**: 紧急情况可以标记
- 📅 **周末**: 回应可能稍慢，请理解

## 🆘 帮助和支持

### 遇到问题？
1. **页面加载问题**: 刷新页面或清除缓存
2. **提交失败**: 检查网络连接，稍后重试
3. **找不到回应**: 检查垃圾邮件或等待24小时
4. **其他问题**: 通过页面反馈功能联系我们

### 紧急情况
如果你或你认识的人处于危机中：
- 🚑 **立即求助**: 拨打当地紧急电话
- 🏥 **专业帮助**: 联系心理健康专业人士
- 👥 **支持网络**: 联系家人、朋友或支持团体

**树洞不是危机干预平台**，但我们可以提供资源指引。

## 🖥️ 第二步：Render部署（看图点击）

### 图4：打开Render网站
```
1. 浏览器打开：https://render.com
2. 用GitHub账号登录
```

### 图5：点击创建服务
```
1. 点击蓝色按钮"New +"
2. 选择"Web Service"
3. 选择我们的仓库
```

### 图6：直接创建
```
1. 名称：secret-treehouse-api
2. 点击"Create Web Service"
3. 等待5分钟
```

## 🎨 第三步：Vercel部署（看图点击）

### 图7：打开Vercel网站
```
1. 浏览器打开：https://vercel.com
2. 用GitHub账号登录
```

### 图8：导入项目
```
1. 点击"Add New"
2. 选择"Project"
3. 选择我们的仓库
```

### 图9：重要配置
```
在"Root Directory"输入：
./public

然后点击"Deploy"
```

### 图11：测试前端
